**Band Sandbox**

Band Sandbox is a realtime live music making enviornment. Users can explore sounds together and practice in real time with a metronome. Users can also sign-up and log-in. 

**Technology Used**

* React JS
* Node JS
* MongoDB
* Express
* Socket.io
